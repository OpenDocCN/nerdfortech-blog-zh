> 原型观念

我将为原型创建三个级别。这个原型的主要目的是展示玩家将经历的一些敌人类型和生物群落。

我们穿着闪亮盔甲的骑士将从森林进入，他的目标是到达城堡，将王国从诅咒中拯救出来。

经过森林后，他将进入一个洞穴，这个洞穴将带他进入已经变成地狱的王国。
