# ICO 是什么？

ICO 代表首次发行硬币。这是一种最新的融资方式，因其独特的联系全球投资者的方式而在公众中日益流行。有想法的项目所有者可以在交易所列出他们的项目，并通过他们的白皮书从投资者那里获得资本。

**初始硬币发行有两种类型**

‣ ***公共 ICO :***

公共 ICO 允许普通大众通过投资来参与项目。任何人都可以购买数字资产。

‣ ***私人 ICO :***

顾名思义，私募 ICO 只允许有限数量的投资者参加筹款活动。它将只允许被认可的人士或基金机构参与。
