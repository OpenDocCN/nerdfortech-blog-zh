给定一个整数numRows，返回帕斯卡三角形的前 numRows。

在帕斯卡三角形中，每个数字是它正上方的两个数字之和，如图所示:

例 1: 

Input: numRows = 5
Output: [[1],[1,1],[1,2,1],[1,3,3,1],[1,4,6,4,1]]

例 2: 

Input: numRows = 1
Output: [[1]]

约束:

  • 1 <= numRows <= 30
class Solution {
public:
    vector<vector<int>> generate(int numRows) {
        vector<vector<int>>r(numRows);
        
        for(int i=0 ; i<numRows ; i++)
        {
            r[i].resize(i+1);
            r[i][0] = r[i][i] = 1;
            
            for(int j=1 ; j<i ; j++)
                r[i][j] = r[i-1][j-1] + r[i-1][j];
        }
        return r;
    }
};
