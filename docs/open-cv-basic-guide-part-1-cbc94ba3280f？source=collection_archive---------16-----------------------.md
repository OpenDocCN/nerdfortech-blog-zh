# 简介:

OpenCV 代表开源计算机视觉库。这是 python 库用来解决计算机视觉问题的。通过使用 OpenCV，我们可以轻松地添加不同的图像，或者有时我们可以裁剪图像，或者我们可以检测图像中的边缘，或者我们可以使用它进行人脸检测等。基本上，我们可以通过使用 OpenCV 库非常容易地执行计算机视觉问题。
