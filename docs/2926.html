4.休息一会儿

很多时候，在 switch 情况下缺少 break 语句直到在生产中运行时才被发现。

如果您忘记了代码示例中 case 0 的一个断点，程序将写“0”后面跟着“1”，因为这里的控制流将遍历整个 switch 语句，直到到达一个断点。因此，在编写开关盒时再次检查。
