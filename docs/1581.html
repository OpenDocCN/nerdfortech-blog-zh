Vue 组合 API:一瞥

这里是对 Vue 3 中发布的新组合 API 的一个 hello-world 级别的介绍。想要更透彻的解释，你可能想参考 Vue 这里的官方文档。

TLDR

Vue 的 composition API 提供了一种以函数式编程风格编写 Vue 组件的方法，通过按目的组织变量和方法，而不是按传统方式(或称为 Options API)的类型。

第一眼

让我们把代码组织的区别放在伪代码中:

选项 API(常规架构)

数据:

-活动 1 的变量 1

-活动 2 的变量 2

方法:

-活动 1 的功能 1

-活动 2 的功能 2

组合 API(来自 Vue 3 的新特性)

活动 1:

-活动 1 的变量 1

-活动 1 的功能 1

活动 2:

-活动 2 的变量 2

-活动 2 的功能 2

当您的 Vue 组件变大并开始包含多个活动时，这样的组织会很有帮助。如果活动是在组合 API 中设计的，则添加或修改活动不需要编辑代码的多个部分。
