# 评估供应商的标准

虽然有几个因素会影响供应商评估，但无论具体行业如何，每个公司都应该解决几个问题。

根据具体情况，具体的供应商评估计划可以在以下情况下为您提供帮助:

*   打算再次购买
*   在一个类别的“真正”供应商中挑选最佳供应商
*   淘汰不合格和表现不佳的供应商，用更好的供应商取而代之
*   启动常规成本效益分析实践

通过定期进行供应商评估，您的公司可以更全面地了解运营情况，降低风险，主动识别增长机会，简化生产时间表，消除不必要的开支，并保持供应商的业绩。
