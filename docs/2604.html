页面对象模型，也称为 POM，是一种创建用于存储所有 web 元素的对象存储库的设计模式。它有助于减少代码重复和改进测试用例维护。在页面对象模型中，将应用程序的每个网页视为一个类文件。每个类文件将只包含相应的网页元素。使用这些元素，测试人员可以在被测网站上执行操作。
