字里行间

你用过行高属性吗？如果我有，我不能马上回忆起来。其中一个评估问题询问该房产的主要用途。可能的答案是:

  • 控制字符间距的宽度
  • 控制字符大小的高度
  • 控制标题元素之间的间距高度
  • 控制两行内容之间的高度

鉴于该属性处理的是高度而不是宽度，您可以很容易地忽略第一个选项。并且考虑到属性处理的是行的高度，而不是字符的高度，第二个选项也可以排除。那么行高只控制标题元素之间的间距吗？Mozilla Developer Network 通知我们它“设置了一个行框的高度”。它通常用于设置文本行之间的距离。因此，最后一个选项将是正确答案，因为行高不仅限于标题元素。顺便说一下，行高的值可以用几种方式表示，比如像素、em和百分比。
