# 简介:

**编程是向计算机提供各种编码指令以执行特定任务的行为**。编程被应用于从电器、商业软件、游戏到医疗设备的方方面面，就在我们说话的时候，它也被应用于医疗设备，比如心脏起搏器和核磁共振成像仪。
