## 并将您的项目存储在 GitHub 存储库中

**目标**:快速为 Unity 项目建立一个 GitHub 库

在本教程中，我将展示如何在短时间内让你的 Unity 项目 GitHub 就绪。

## 步骤# 1——创建一个 Unity 项目

我建议用 Unity Hub。在项目视图中，只需点击**新建**，然后在你的电脑上选择一个模板、一个名称和一个位置。然后**创造**。
