> 标签用于在文档中嵌入声音内容，比如音乐或其他音频流。标签包含一个或多个具有不同音频源的标签。浏览器将选择它支持的第一个信号源。标签和之间的文本只会在不支持元素的浏览器中显示。HTML 支持三种音频格式:MP3、WAV 和 OGG。不要忘记设置控制，否则您的音频将不会加载到网页上。
