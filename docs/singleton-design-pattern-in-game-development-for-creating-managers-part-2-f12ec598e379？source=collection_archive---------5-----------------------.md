编写单例类的方式非常简单。

*   创建一个私有的静态实例变量来保存这个类的唯一对象。
*   根据需要创建 get、set 属性。
*   使用唤醒调用初始化它。

除了这三个基本需求，这个类还可以有任意多的其他属性和方法。
