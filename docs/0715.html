问题:求给定数组中最大和最小的数。

让我们使用之前看到的相同数组。

方法:

为了找到最大的和最小的，我们几乎应用相同的方法。

我们将假设我们的第一个元素是最小/最大的，并将其存储在变量 X 中。然后，我们转到其他节点，查看它们是否小于/大于我们的变量 X 中存储的值。如果它们通过检查，则将它们存储在变量 X 中，或者如果失败，则移动到下一个，重复此操作，直到到达末尾。
