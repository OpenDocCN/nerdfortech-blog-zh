配置文件系统

现在让我们在文件系统上运行 fsck 命令&用 resize2fs 命令调整文件系统的大小。

  • e2fsck 用于检查 ext2/ext3/ext4 系列文件系统。
  • resize2fs 是一个命令行实用程序，允许我们调整 ext2、ext3 或 ext4 文件系统的大小。
>> resize2fs -f /dev/sdh1

重新挂载扩展分区

>> mount /dev/sdh1 /Partition
