“移动零”是一个 leetcode 挑战，提示您将数组中的所有零移动到该数组的末尾。

# 问题是:

给定一个*num 的数组，*写一个函数将所有的 0 移动到它的末尾，同时保持非零元素的相对顺序。

## 示例:

```
**Input:** [0,1,0,3,12]
**Output:** [1,3,12,0,0]
```
