# **练习 01**

**计算列表条目的数量，并添加一个 para 标签来显示结果。**

**超文本标记语言**

```
<body>
<ul>
<li>Apple 🍎</li>
<li>Grapes 🍇</li>
<li>Pear 🍐</li>
<li>Orange 🍊</li>
</ul>
<input type="button" onclick="getCount()" value="Count all items in the list">
</body>
```

# **练习 03**

***显示窗口的宽度和高度(当窗口调整大小时应该会改变)***

**超文本标记语言**

```

</body>
```
