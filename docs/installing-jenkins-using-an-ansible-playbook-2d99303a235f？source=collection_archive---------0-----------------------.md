## 詹金斯

Jenkins 是一个独立的开源自动化服务器，用于自动化与构建、测试和交付/部署软件相关的任务。Jenkins Pipeline 通过使用插件和一个 Jenkinsfile 实现了向 Jenkins 的连续交付管道。Jenkinsfile 可以是声明性的，也可以是脚本化的，并且包含管道要遵循的步骤列表。
