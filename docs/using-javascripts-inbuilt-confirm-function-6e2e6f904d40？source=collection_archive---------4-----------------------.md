# 应用

在您的表单中，您可能希望用户使用事件监听器 **onsubmit** 并传递 **return confirm("您确定吗？")**。

现在，这将提示用户消息*您确定要提交吗？*。如果用户提供了肯定的回答，表单就可以提交，但是如果用户提供了否定的回答，表单就不能提交。
