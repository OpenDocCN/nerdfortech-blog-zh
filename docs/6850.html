撰写中的按钮介绍

按钮是 Android 中最常用的 UI 元素。几乎 80 %的动作都是由按钮发起的。按钮包含文本，可点击的属性，图像等。

在材质设计 3 中，我们可以有 5 个版本的按钮作为提升、填充、填充色调、轮廓&文本。

注意:材质设计是一个适应性很强的指导方针、组件和工具系统，支持用户界面设计的最佳实践。材料设计简化了设计师和开发人员之间的协作，并帮助团队快速构建漂亮的产品。

我们已经有一个材料设计 2 按钮规格。但是最近，材料设计 3 发布了。看看上有什么新消息

所有按钮类型的一般信息和语法类似于👇

高架按钮T20】带阴影的基本轮廓按钮。仅在需要与背景分离时使用。

装满按钮
和装满容器按钮。它的对比色表面使它成为最突出的按钮

轮廓按钮
它们多用于非主要动作。这是一种按钮，我们描绘用户给一些机会来执行动作或从流中逃脱

文本按钮
这是一种当我们想把动作描绘得不那么引人注目时使用的按钮。当我们向用户展示多个选项时，我们可以考虑使用这种类型的按钮。

FilledTonalButton
T32】这是浅色背景&深色标签颜色。它们可以用来描述不如填充按钮那样引人注目的动作。

这是最基本的🙄，我们还可以用按钮参数做什么🤔
在每个按钮的可组合语法中，都有一些公共参数，您可以使用它们来满足您的用例，例如

  1. 启用/禁用按钮
    这可以使用启用参数来控制
  2. 设置仰角以区分按钮和背景
    这可以使用仰角参数来控制
  3. 为按钮
    设置不同的形状，这可以使用形状参数进行控制
  4. 设置按钮
    的边框颜色，这可以使用边框参数进行控制
  5. 设置不同的背景/内容/禁用/启用颜色
    这可以使用颜色参数进行控制
  6. 在按钮中传递不同的文本和图像(以行的方式)
    这可以使用内容参数来控制
