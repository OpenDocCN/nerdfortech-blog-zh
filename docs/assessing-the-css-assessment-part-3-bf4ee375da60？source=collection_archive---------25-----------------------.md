**选择选择器**

学习 CSS 的第一个主题是[选择器](https://www.w3schools.com/cssref/css_selectors.asp)，因为它们使你能够选择设计哪些元素。CSS 选择器有多种，包括元素、类、id、属性、伪类和伪元素。评估中的这个问题应该不会给 CSS 新手带来任何困难。

*下面的选择器匹配什么元素？*

1.  *。导航{…}*
2.  *导航{…}*
3.  *#nav {…}*

虽然这三个例子都在选择器中包含了 *nav* ，但是每个例子都有一个很小但很重要的区别，这表明它不会像其他两个例子一样应用于相同的元素。第一个以“.”开头，表示类选择器。第三个以“#”开头，表示 id 选择器。第二个是元素选择器，前面什么也没有。因此，您可能会毫不费力地发现问题的正确答案是，第一项引用了一个类为“nav”的元素，第二项引用了一个 nav 元素，第三项引用了一个 id 为“nav”的元素。

你知道类选择器和 id 选择器有什么不同吗？您可以将一个类应用于 web 页面上任意数量的元素，而一个 id 只能应用于页面上的一个元素。此外，一个元素可以有多个类(您可能会在评估中看到一个关于这个的问题—提示，提示)。

另一个问题是关于一个特定的[伪类](https://developer.mozilla.org/en-US/docs/Learn/CSS/Building_blocks/Selectors/Pseudo-classes_and_pseudo-elements)选择器——*:active*伪类。您可能会经常看到 *:active* 应用于链接，但是您可以将它用于任何其他元素。该问题呈现以下风格规律:

```
a:active { color: pink;}
```

在本例中，您希望将锚定标签的文本更改为粉红色，但是什么时候会发生呢？是在链接被点击之前，在鼠标悬停时，在链接被点击时(但在鼠标点击被释放之前)，还是在它被点击之后？顾名思义， *:active* 伪类在链接活动时生效——也就是说，当您单击链接时生效，并且只要您按住鼠标单击，它就一直有效。一旦释放鼠标点击，链接不再激活，并且*:激活*样式规则不再适用。

顺便说一下，链接的其他伪类包括 *:link* 、 *:visited* 和 *:hover* 。

还有一个选择题…

考虑以下可能的答案:

*   一个[标题] {…}
*   a >标题{…}
*   a.title {…}
*   a-title {…}

上面哪个选项具有选择带有“title”属性的元素的正确语法？快速复习:根据 W3Schools.com 的说法，HTML [属性](https://www.w3schools.com/html/html_attributes.asp)s——比如*标题*、 *href* 、 *src —* “提供关于 HTML 元素的附加信息。” [title](https://developer.mozilla.org/en-US/docs/Web/HTML/Attributes) 属性创建“当鼠标悬停在元素上时在工具提示中显示的文本”谢天谢地，这又是一个简单的。CSS [属性选择器](https://www.w3schools.com/css/css_attribute_selectors.asp)用方括号括起来；因此，第一个选择是正确的。

**喜欢那些线条高度**

在[之前的帖子](/nerd-for-tech/assessing-the-css-assessment-part-1-2fae27b92e2)中，我提到了一个关于评估的问题，询问关于 [*行高*](https://developer.mozilla.org/en-US/docs/Web/CSS/line-height) 属性的用途。答案是，它控制两行内容之间的空间高度。但是你能辨别行高设置“20px”和行高设置“2”之间的区别吗？所有四个可能的答案都表明，20px 的设置意味着行高将被设置为…你猜对了，20px。2 的设定怎么样？这是有效值吗？它会默认为 2px 值吗？它是将行高设置为相应字体大小值的两倍，还是相应字体大小值的 20%呢？

行高属性可以有四个值:关键字*普通*，默认值；一个数字(后面没有单位)；长度，以单位表示(例如，px、pt、rem、em、VH)；和一个百分比。因此，将该属性的值设置为 2 是可行的，现在的问题是知道该值意味着什么。将该属性单独表示为一个数值与 px 之类的单位无关，也与当前字体大小的百分比无关。因此，答案是，值 20px 将行高设置为 20px，值 2 将行高设置为相应字体大小值的两倍。
