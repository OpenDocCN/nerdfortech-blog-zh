# k 均值聚类

![](img/0e9545a9985ecaca04ae7655bf72ac8c.png)

图上的簇和质心

如果一个**聚类**被定义为由于某些相似性而被分组在一起的数据点的选择，那么**k-均值聚类**可以被定义为在数据集中寻找固定数量的聚类 ***k*** 的方法。对数据点进行聚类可以帮助我们指出数据中的趋势，否则我们可能无法检测到这些趋势，并且可视化它们是如何分组在一起的，可以让我们更清楚地了解整个集合。默认情况下，K-means 聚类依赖于**欧几里德距离**来确定相似性——这是一种基于欧几里德空间或几何平面中的点的实际距离的度量。该平面上簇的位置由其**质心**决定，质心是代表每个簇中心的特定点。使用 k-means 聚类的关键原因？虽然我们可以任意决定任何数量的质心或聚类来测量我们的数据集，但有些数量比其他数量更优。[【4】](https://towardsdatascience.com/understanding-k-means-clustering-in-machine-learning-6a6e67336aa1)

![](img/c52625d6a4dd99f9ac2e696d136f5774.png)

肘法

为了确定聚类的最佳数量 *k* ，我们可以使用一种流行的启发式方法，称为**肘方法，**该方法针对一系列 *k* 值对数据集运行 k 均值聚类，计算每个值的误差平方和(SSE)，并将 SSE 绘制在一条线上。虽然我们确实想要一个相对较小的 SSE，但是随着我们增加 *k，*，SSE 也向 0 减小，并且只有当簇的总数等于点的总数时，SSE 才达到 0。因此，最好的妥协是在中间的某个地方，这条线开始从一个极端方向弯曲到另一个极端方向。然后，该点被命名为图形的**弯头**。[【5】](https://www.geeksforgeeks.org/elbow-method-for-optimal-value-of-k-in-kmeans/)

阿尔·戈尔，美国第 45 任副总统，2000 年大选的前总统候选人，世界著名的环境保护主义者。我无法确定阿尔·戈尔出现在这么多名单上的确切原因(我也无法访问这些名单本身)，但我可以想象这与气候变化有关——这是他最著名的话题。然而，他的列表与关注者的比率完全是一个异常，在没有更多信息的情况下很难解释。

# 限制

在执行 k-means 聚类时，我观察到的第一个限制是聚类如何如此容易受到离群值的影响——这种情况下的主要例子是巴拉克·奥巴马，他的追随者数量如此之高，以至于盖过了所有其他人，导致他有了自己的聚类。在这种情况下，很难分析其他集群，因为它们之间的差异相比之下完全相形见绌。他必须完全从图表中删除，以便正确地测量其他细节。

这里的另一个限制是，我不能对非数字信息进行聚类，比如这个人的性别或政党。了解这两个因素中的任何一个是否与这个人在网上的受欢迎程度有关，或者他们在社交媒体上的表现如何，这将是一件有趣的事情。

如果这个数据集包括前总统唐纳德·特朗普(Donald Trump)也很有趣，他在 Twitter 上的活动非常独特，如果他没有被禁止，几乎肯定会成为图表上的另一个异常值。在 2021 年 1 月 8 日被永久删除之前，该男子拥有近 90，000，000 名粉丝和 6 万条推文。[【7】](https://www.tweetbinder.com/blog/trump-twitter/)

值得一提的最后一个细节是，Twitter(和一般的大型科技公司)倾向于在政治光谱上左倾，至少在社会政策方面。 [[8]](https://www.washingtonpost.com/technology/2018/08/19/twitter-ceo-jack-dorsey-admits-) 虽然其首席执行官杰克·多西声称他个人的偏见不会影响公司的政策，但该公司的实际服务条款主要反映了自由派的感受——特朗普当选总统后相对较新的用户禁令浪潮对共和党人、保守派和右翼的影响可以说超过了政治光谱的另一边。[【9】](https://www.usatoday.com/story/opinion/2021/01/14/trump-banned-twitter-rachel-bovard-column/4145532001/)结果，许多与这些意识形态团体结盟的用户放弃了 Twitter，在其他限制较少的平台上寻求庇护，如 Parler 或 Gab。根据这些信息，民主党政客在推特上更受欢迎是否是社交媒体外流的部分结果？还是民主党政客只是对科技更友好？如果我们分析其他社交媒体网站，我们会发现什么样的趋势？当政治人物转移到其他平台时，他们的行为和由此产生的统计数据会发生变化吗？这些都是有趣的问题，没有更多的数据我们根本无法回答。

# 结论

在做了这个分析后，我们可以给一个人的一个明显的建议是，要成为社交媒体上受欢迎的政治家，你不一定要年轻。

此外，虽然某些杰出的人，如巴拉克·奥巴马、唐纳德·特朗普、AOC(甚至可能还有坎耶·韦斯特)因利用强大的社交媒体存在来支持他们的个人成功而臭名昭著(尽管这仅适用于坎耶的音乐生涯，而不是他的政治竞选)，但他们并不是人们可能认为的多产海报。

事实上，*最受欢迎的*政客倾向于发布*最少的*，并且与其他人发布的帖子互动较少，这一悖论也暗示了社交媒体上的受欢迎程度不仅仅是某人在网上有多少活动的产物，而是他们*在他们的帖子中说了什么*，以及他们*在现实世界中做了什么*才值得开始发布。这也可能意味着他们只是太忙于在现实世界中说和做重要的事情，而没有时间不断更新他们在 Twitter 上的状态。

另一方面，数据集中一些最不受欢迎的政治家似乎有意通过大量的活动来吸引注意力——不幸的是，这种策略不太可能奏效，至少基于这一数据。

那么，这个故事的寓意是什么？或许在社交媒体上成为一个受欢迎的政治家，不在于你的朋友数量或你的帖子数量，而在于你的想法的价值和你的性格品质。

(或者，其他一些我尚未确定的更模糊的统计数据。)
