# 在 Unity 中享受基础物理的乐趣

游戏引擎中的物理工作很像现实生活中的物理。在奇妙的物理世界中，物体**相互作用**，有运动，有质量，并屈服于重力之类的东西！

要在 Unity 中使用物理引擎，你需要两个组件:**对撞机**和至少一个**刚体**。

游戏中有 **2D** 和 **3D** 版本的碰撞器和刚体…
