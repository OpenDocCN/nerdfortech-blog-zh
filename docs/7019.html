Dockerfile 的最佳实践是保持图像最小化。创建一个分布式映像，避免包含不必要的包或暴露端口，以减少攻击面。

分布式图像

这篇文章包含了如何建立一个最小的 Docker 镜像的说明，这个镜像只有一个tmp文件夹，它是一个“分布式镜像”
