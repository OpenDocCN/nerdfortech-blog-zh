# SEO 优化

Create React App 在服务器端运行应用程序，对于谷歌来说，抓取这些网站的内容相当困难。Google 对静态文件的抓取最好，其次，JS 确实提供了静态文件并负责处理静态文件。Next JS 帮助你的网站更容易被 google 搜索爬虫抓取，从而提供更好的 SEO。
