# 为什么是自主持跑步者？

> *与 GitHub 托管的 runners 相比，自托管的 runners 非常适合在一个高度可定制的环境中运行工作流，对硬件要求、安全性、操作系统和软件工具进行更精细的控制。*
> 
> *自托管运行者可以是物理的、虚拟的、在容器中的、内部的或云中的。在本指南中，我们将把它作为一个容器部署在 AWS 云中的 Kubernetes 集群中。*
