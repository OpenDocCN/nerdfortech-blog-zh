# 链接列表

> 原文：<https://medium.com/nerd-for-tech/linked-lists-72369461d53d?source=collection_archive---------20----------------------->

在编程中，有几种不同的数据结构，每种都有自己的优点和缺点。其中一种数据结构叫做链表，在链表中，有单向链表和双向链表。顾名思义，两者的区别在于单链表在两个单向的节点之间有一个链接。而双向链表在节点之间有两个链接，一个链接指向下一个节点，另一个链接指向前一个节点。下面是这两者之间的区别:

![](img/1fa9c179373cbf0f039420b81cf63b9c.png)
