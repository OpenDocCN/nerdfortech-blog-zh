好了，在你燃起希望之前，预制画笔已经不见了。从 2D TileMap Extras 1.6 或更高版本和 Unity 2020 或更高版本开始，你不能再在Unity Tile调色板中使用预设笔刷。然而，并没有失去一切。虽然预制笔刷已经被移除，但是有一个替代笔刷存在。

从 2D TileMap Extras 1.6 和更新版本开始，你现在必须使用游戏对象笔刷来获得大部分预设笔刷功能。
