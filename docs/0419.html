对于作用在楔形土壤上的浅基础和深基础非常重要。给出了楔形土所经历的粗略的比喻应力分布的 Python 实现。

地基: 它是一个结构的最低部分。

扩展基脚: 是承重墙或承重柱的扩大，使得将结构的负荷分散到更大面积的土壤上成为可能。

垫式基础: 低承载能力的土壤要求扩展基脚的尺寸过大。在混凝土垫层上建造整个结构更经济。

扩展基础和垫式基础通常被称为浅基础，而桩和钻孔基础被归类为深基础。

为什么方程会因为立足点而改变？
