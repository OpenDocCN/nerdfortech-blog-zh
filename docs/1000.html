React JS是用来创建用户界面的最流行的Javascript库之一。它是由脸书开发的。React不是一个框架，它是一个库，因为它专注于视图层。

React为我们提供了许多优势。例如，任何网站的中心都有一个关于页面如何组织的结构，称为面向文档的模型(DOM)。每次用户刷新网站时都会更新DOM，导致高用户网站的性能下降。React使用虚拟DOM只渲染有变化的地方。此外，React使用JSX，它可以用HTML风格的代码修改和开发自己的DOM。JSX是一种允许用Javascript编写HTML的格式，对于网站性能是一种非常有效的解决方案。React是基于组件的，因此允许我们避免重复，用一个函数获得多个解决方案。

在简单描述了React JS之后，让我们安装需求并使用React编写我们的第一个应用程序。
