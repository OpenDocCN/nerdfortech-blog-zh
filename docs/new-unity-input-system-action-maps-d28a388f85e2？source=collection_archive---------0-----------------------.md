# 新的 Unity 输入系统:动作地图

![](img/71028644c97c8a78e34fbfe71277b1a4.png)

现在已经为*玩家*创建了*输入动作资产*，我们可以更深入地创建*动作地图*。

打开*输入动作资产*，注意窗口左上角的*动作映射*区域。按下*加*T12(+)图标，创建一个新的*动作地图*。我创建的第一个将被称为*玩家*，这将是主要玩家控制的动作地图。
