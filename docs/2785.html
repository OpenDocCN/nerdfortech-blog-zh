冒泡排序算法的实现。

简介:

  • 冒泡排序用于对给定数组中的元素进行排序。
  • 它通常在数组很小的时候使用。
  • 当列表部分排序时，它工作得很好。
  • O(n)是冒泡排序的时间复杂度。

冒泡排序是如何工作的？？

让我们明白…

它在每次迭代中比较相邻的值，如果它们的顺序错误(第一个值大于第二个值),则交换它们。

在每次迭代中，较大的值放在列表的末尾，(len(list)-i)。

这里‘I’是第I次迭代。

实施:

def bubble_sort(arr):
    for j in range(len(arr)-1):
        swapped = False
        for i in range(len(arr)-1-j):
            if arr[i]>arr[i+1]:
                arr[i], arr[i+1] = arr[i+1], arr[i]
                swapped=True

  • 如果我们的算法已经排序，那么交换值为假，它打破了循环。
  • 或者在第n次迭代中，列表被完全排序，然后在该迭代之后，交换的值为假，并且它中断循环。
