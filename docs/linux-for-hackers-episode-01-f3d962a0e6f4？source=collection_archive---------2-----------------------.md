# 摘要

*   Linux 是内核，不是 OS(操作系统)
*   Kali Linux、Parrot OS、Ubuntu 等都是 OS 或 Linux 版本或发行版。
*   内核是应用程序、操作系统和硬件之间的中间人。
*   Linux 是开源的，速度更快，也更安全。
