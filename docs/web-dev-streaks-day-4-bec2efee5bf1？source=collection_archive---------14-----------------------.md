> 由于编写代码、保存代码，然后通过在 web 浏览器中重新加载页面来查看更改的任务相当枯燥，因此可以通过在 VS 代码中安装一个 live server 来进行排序(在扩展中搜索“live server”，然后安装)。注意，回到你的“index.html”文件，点击“Alt+L”然后“Alt+O”一个新的页面将会在你的浏览器中打开。
