# 在后台

加载器是在获取数据时将用户保持在平台上的好方法。它总是增强用户体验，并且易于集成到任何应用程序中。如果你读过我以前的帖子，在几乎每一个 API 获取过程中，我都会在获取数据时添加加载器或旋转器。
