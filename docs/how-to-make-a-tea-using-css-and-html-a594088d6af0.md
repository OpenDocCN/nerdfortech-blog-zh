## 什么是 CSS 动画？

动画让元素逐渐从一种风格变成另一种风格。

您可以根据需要多次更改 CSS 属性。

要使用 CSS 动画，您必须首先为动画指定一些关键帧。

关键帧保存元素在特定时间将具有的样式。
