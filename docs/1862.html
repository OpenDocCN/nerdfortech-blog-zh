我将使用非常基本的Titanic数据集，但替换数据集只是2行代码的更改。

假设

  1. 您已经清理了数据集
  2. 变量被转换成正确的数据类型。

一旦这些步骤完成，使用dash我们可以设置一个简单的基于浏览器的界面来进行单变量分析。

装破折号 

pip install dash dash-renderer dash-html-components dash-core-components plotly
