# 修复背景后面的精灵

这很容易被忽视，尤其是如果你不熟悉 2D 的工作流程。2D 精灵是在**排序层**上**书写/渲染**的，它们的**顺序**在那层。

就像蛋糕的各层一样，糖衣是在实际蛋糕之前看到的——首先看到的是**更高的层**，而**更高的数字顺序**是该层中首先看到的。
