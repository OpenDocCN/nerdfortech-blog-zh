什么是 Appium automation？

Appium 是一个神奇的测试自动化框架，它是所有本机、混合、Android、iOS 和 Windows 应用程序的朋友。这意味着 Appium 不会歧视和自动测试所有类型的 Flutter 应用程序。此外，由于 Appium 是跨平台的，它允许使用相同的 API 为不同的平台编写测试。这难道不值得称赞吗？

这里有一些使用 Appium 的优点:

  • 允许编写和运行测试以确保质量
  • Appium 支持多种平台和操作系统
  • 不需要对应用进行任何重新编译
  • 支持 JSON 写协议
  • 支持物理设备上的自动化测试
  • Appium 对移动设备没有任何依赖性
