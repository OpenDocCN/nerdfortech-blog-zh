在成为Flutter开发人员之前，我来自JavaScript世界。我最怀念的一件事是npm上强大的JavaScript包，尤其是lodash和ramda，它们为我们访问嵌套的JSON对象提供了极大的便利。

在开发Flutter应用程序时，我们经常需要从API响应中解析JSON对象，然后相应地访问或操作它们。遗憾的是dart语言对Map对象的支持不是很丰富，因此我创建了这个包:map_enhancer。

用途

实际上这个包并不局限于Flutter和JSON(Map<String, dynamic>)对象，你可以在任何dart项目中使用它，或者在任何dart的本地Map对象上使用它。

仅此而已。访问 pub.dev 来安装这个包，或者在 GitHub 上提交问题/请求。
