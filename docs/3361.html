现在让我们看看什么是超平面。超平面的概念只不过是直线概念的延伸。是的，就是这么简单，你很快就会明白怎么做了。首先，让我们看看直线的一般方程。我们知道 2D 一条线的一般方程是:y=mx+c，从高中数学我们知道这个方程来自 ax+by+c=0。这个等式也可以写成 y= -(c/b) +(-a/b)x，其中(-a/b)形成直线的斜率‘m’，而(-c/b)是 y 截距。因此，你可以清楚地看到上述两个方程之间的相似性。现在，如果轴是 x1 和 x2 而不是 x 和 y，等式也可以写成 w1x1+w2x2+w0=0，其中 w1 和 w2 只不过是 x1 和 x2 的系数，并且在 ax+by+c=0 的情况下与“a”和“b”相同。还有，w0 只是一个常数。
