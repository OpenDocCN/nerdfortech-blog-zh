*   实验项目|第三部分，在课程的这一部分，将会教你关于画布的知识。以及如何在里面编辑东西。比如改变颜色、大小等等。
*   银河计划，如果你和你自己一起遵循这个过程。你将能够很容易地用 Python Kivy 框架制作应用程序。
